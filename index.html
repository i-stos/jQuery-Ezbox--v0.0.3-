Ezbox_v0.0.3 A simple jQuery overlay

Required Files

ezbox.css, jquery.min.v1.7.1.js, ezbox.js

Install

1. Add the ezbox.css file, ezbox.js, and create a

tag, or any other block element with a class name of 'ezbox' like so:

2. Inside this block element insert your thumbnail images and the links that point to the larger images in the following format:

3. Call Ezbox with the desired settings like so:

  $('.ezbox').EzBox({
      overlayBg: "#262626",  //overlay background color
      overlayOpacity: 0.58, //overlay background opacity
      scrollAllowed: true //when true the overlay scrolls as the user scrolls, when false it doesn't
  });
